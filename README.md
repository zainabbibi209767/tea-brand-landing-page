🍵 Green Tea Landing Page

A modern and responsive green tea landing page UI concept designed for tea brands, cafés, and beverage businesses.

## ✨ Features

- Elegant Tea UI Design
- Responsive Layout
- Premium Beverage Showcase
- Smooth Typography
- Fresh and Natural Theme
- Call-to-Action Buttons
- Modern Landing Page Design

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript

## 📂 Project Setup

Clone the repository:

```bash
git clone https://github.com/your-username/green-tea-landing-page.git
```

Open the project folder:

```bash
cd green-tea-landing-page
```

Run the project by opening `index.html` in your browser.
